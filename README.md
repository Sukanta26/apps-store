# Apps Store

## Description

Apps Store is a responsive web application that showcases trending apps with details like name, company, description, downloads, ratings, and reviews. Users can search for apps and view information in a clean, modern, and interactive interface. This project uses JSON data to dynamically display app information.

## Features

- Display a list of apps with images, titles, companies, descriptions, ratings, and downloads.
- Responsive grid layout for mobile, tablet, and desktop views.
- Search functionality to filter apps by name.
- Hover effects and clean UI using Tailwind CSS.

## Technologies Used

- **React.js** – Frontend library for building UI components.
- **Tailwind CSS** – Utility-first CSS framework for styling.
- **daisyUI** – Tailwind CSS component library (optional for additional UI components).
- **JavaScript (ES6+)** – Logic and interactivity.
- **HTML5 & CSS3** – Basic markup and styling.
- **JSON** – Static data source for app details.
